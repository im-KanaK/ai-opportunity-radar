# 🚀 AI Opportunity Radar

An AI-powered automation workflow that discovers opportunities, matches them with user interests, and sends personalized recommendations automatically.

## Features

* Fetches opportunities from Devpost
* Fetches opportunities from Unstop
* Stores opportunity data in Supabase
* Matches opportunities with user skills and interests
* Generates opportunity scores
* Selects top opportunities
* Creates personalized HTML email digests
* Sends automated Gmail notifications

## Tech Stack

* n8n
* Supabase
* JavaScript
* Devpost API
* Unstop API
* Gmail
